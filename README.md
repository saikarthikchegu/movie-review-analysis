# 🎬 Movie Review Sentiment Analysis using Machine Learning

## 📌 Project Overview
This project performs sentiment analysis on movie reviews using Machine Learning. It classifies reviews as either **Positive** or **Negative** by applying Natural Language Processing (NLP) techniques and a Logistic Regression classifier.

The model uses TF-IDF vectorization to convert text into numerical features before training the classifier.

---

## 🚀 Features
- Load and preprocess the IMDB movie review dataset
- Clean and normalize text data
- Remove stopwords
- Convert text into numerical vectors using TF-IDF
- Train a Logistic Regression model
- Predict review sentiment
- Evaluate the model using:
  - Accuracy Score
  - Classification Report
  - Confusion Matrix
